# Project Code Management (PCM)

A centralized system for managing all projects' code, versioning, documentation, and CodeGraph.

## Overview

PCM serves as the single source of truth for:
- **Code Management**: Track and organize source code across all projects
- **Version Control**: Manage versions, branches, and releases
- **Documentation**: Centralized docs and knowledge base
- **CodeGraph**: Visualize and analyze code relationships

## Directory Structure

```
pcm/
├── src/           # Source code
├── docs/          # Documentation
├── scripts/       # Automation scripts
└── README.md      # This file
```

## Getting Started

```bash
npm install
npm run dev
```
